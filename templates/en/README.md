# English Templates

English templates will be generated by copying the Japanese templates and translating the prose content.

To generate English templates from the Japanese originals:

```bash
# Copy structure
cp -r templates/ja/ templates/en/

# Then manually translate the Japanese prose in each .tmpl file
# Command names, variable placeholders, and code remain unchanged
```

For now, use `--lang ja` with init.sh, or manually adapt the Japanese templates.
