# 規約マッピング

変更対象ファイルのパスに基づき、該当する規約を特定して読み込む。

## パス → 規約ファイル対応表

CLAUDE.md に定義された規約マッピングテーブルを参照すること。

規約マッピングは、変更対象パスに対してどの規約ファイルを読むべきかを定義する。CLAUDE.md または `.claude/rules/` を確認して該当する規約を特定すること。

### i18n対応の場合

## i18n 対応（画面変更時）

画面・コンポーネントを変更する場合:
- ハードコード文字列は `プロジェクトのi18nアクセサ（CLAUDE.md 参照）` 経由に置き換える

### コード生成がある場合

## コード生成モデル変更時

コード生成対象のモデルを追加・変更した場合:
```bash
プロジェクトのコード生成コマンド（CLAUDE.md 参照）
```
